# [Phoghorn](https://phoghorn.org)

Walk your classes without walking your classes.

Visit the live site: [Click here](https://phoghorn.org).

Phoghorn is a web app focused on indoor campus navigation with an interactive 3D experience.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- Three.js
- React Three Fiber
- React Three Drei
- GSAP

## Getting Started

### Prerequisites

- Node.js 18+ (Node.js 20 recommended)
- npm

### Install

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Lint

```bash
npm run lint
```

## Project Structure

```text
src/
  components/          Reusable UI and 3D components
  pages/               Landing and application-level pages
  assets/              Models, media, and static design assets
```
